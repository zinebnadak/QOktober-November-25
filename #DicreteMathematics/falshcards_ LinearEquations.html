Linear Equations Flashcards

Q1: General equation of a straight line?
y = kx + m, where k = slope, m = y-intercept.
Q2: How to calculate the slope k given two points?
k = (y2 - y1) / (x2 - x1)
Q3: How can you tell if two lines are parallel?
They have the same slope: k1 = k2.
Q4: How can you tell if two lines are perpendicular?
The product of their slopes is -1: k1 * k2 = -1.
Q5: What if k > 0, k = 0, or k < 0?
k > 0 → line rises
k = 0 → line is horizontal
k < 0 → line falls
Q6: Can a vertical line be written as y = kx + m?
No. Vertical line: x = constant (parallel to y-axis).
Q7: What is a linear system of equations?
Two or more linear equations that must be true at the same time.
Q8: How to solve a linear system graphically?
Draw lines for each equation and find intersection point → solution.
Q9: How many solutions can a linear system have?
One solution → intersect once, none → parallel, infinite → coincide.
Q10: What is the substitution method?
Solve one equation for one variable and substitute into the other equation.
Q11: Solve y = 2x + 4 and y = 3x + 2 by substitution
2x + 4 = 3x + 2 → x = 2 → y = 8
Q12: What is the addition (elimination) method?
Add or subtract equations to eliminate a variable, possibly after multiplying one equation.
Q13: Solve y = 2x + 4 and y = 3x + 2 using addition
Multiply second by -1, then add: x = 2, y = 8
Q14: Why might the graphical method not be exact?
Drawing or reading the graph may give an approximate solution, not exact values.
Q15: When does a linear system have infinitely many solutions?
When lines coincide (same slope and y-intercept).
Q16: When does a linear system have no solution?
When lines are parallel and never intersect.
Q17: What is the intersection point of two lines?
The point where the lines meet; coordinates satisfy both equations.
Q18: How do you check if a solution is correct?
Substitute the solution into both equations to verify equality.
Q19: Can any system be solved using substitution or addition?
Yes, both methods give the same solution if it exists.
Q20: What should you always do first in substitution?
Solve one equation for one variable before substituting into the other.
