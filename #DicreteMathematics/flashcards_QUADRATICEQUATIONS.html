Quadratic Equations Flashcards

Q1: What is a quadratic polynomial?
A polynomial of degree 2, written as ax² + bx + c, where a ≠ 0.
Q2: What determines the degree of a polynomial?
The term with the highest exponent of the variable.
Q3: Give an example of a quadratic polynomial.
Example: x² + 3x + 1, where a = 1, b = 3, c = 1.
Q4: What is a quadratic function?
A function written as f(x) = ax² + bx + c, a ≠ 0.
Q5: What is a quadratic equation?
An equation where one side is a quadratic polynomial and the other is 0: ax² + bx + c = 0.
Q6: What does it mean to “solve” a quadratic equation?
To find x-values that make the equation equal to zero.
Q7: What shape does a quadratic function form?
A parabola.
Q8: What if a > 0 in f(x) = ax² + bx + c?
Parabola opens upward and has a minimum point.
Q9: What if a < 0 in f(x) = ax² + bx + c?
Parabola opens downward and has a maximum point.
Q10: What is an extremum?
The highest or lowest point of a quadratic function (maximum or minimum).
Q11: What is a “happy mouth” parabola?
A parabola with a > 0; opens upward, minimum point.
