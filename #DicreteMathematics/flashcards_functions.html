Functions Flashcards

Q1: What is a function?
A function is a rule that gives exactly one y-value for every x-value in its domain.
Q2: What is the domain and range of a function?
Domain (Df) = set of allowed x-values; Range (Vf) = set of y-values produced.
Q3: How do you write the mapping between domain and range?
f : Df → Vf
Q4: Condition for a relation to be a function?
Each x in the domain corresponds to exactly one y in the range.
Q5: Is x² + y² = 1 a function?
No, some x-values have two y-values — not a function.
Q6: What does f(x + 2) mean?
Replace every x in the rule with (x + 2).
Q7: How to find the domain of a rational function?
Exclude all x-values that make the denominator zero.
Q8: What is an even function?
f(−x) = f(x) for all x in its domain.
Q9: What is an odd function?
f(−x) = −f(x) for all x in its domain.
Q10: Symmetry of an even function?
Symmetric with respect to the y-axis.
Q11: Symmetry of an odd function?
Symmetric with respect to the origin.
Q12: What is a composite function?
(g ∘ f)(x) = g(f(x)) — one function operates on another.
Q13: How to find the domain of a composite function?
All x where f(x) is defined and f(x) lies in g's domain.
