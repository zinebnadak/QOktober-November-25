Set Theory Flashcards

Q1: What is a set and an element?
A set is a collection of objects that share something in common. An element is an object that belongs to a set.
Q2: How can you describe a set?
1. Text: "A is the set of numbers greater than 3"
2. Enumeration: A = {3,4,5,…}
3. Set-builder: A = {x | x ≥ 3, x ∈ Z}
Q3: How do you show that an element belongs to a set?
x ∈ A → x is in set A
x ∉ A → x is not in set A
Q4: What is a subset?
B ⊆ A → all elements of B are in A
B ⊂ A → B is a proper subset of A (A has more elements)
Q5: What is cardinality?
The number of elements in a set. Example: A = {1,2,5} → |A| = 3
Q6: What are natural numbers?
N = {0,1,2,3,…} → non-negative integers
Q7: What are integers?
Z = {...,-2,-1,0,1,2,...} → includes negatives, zero, and positives
Q8: What are rational numbers?
Q = numbers of the form a/b where a,b ∈ Z and b ≠ 0
Q9: What are real numbers?
R = all rational and irrational numbers (like √2, π, e)
Q10: What are complex numbers?
C = numbers of the form a + bi, where a,b ∈ R and i² = -1
Q11: What is the union of sets?
A ∪ B = all elements in A or B or both
Example: A = {1,2}, B = {2,3,5} → A ∪ B = {1,2,3,5}
Q12: What is the intersection of sets?
A ∩ B = elements in both A and B
Example: A = {1,2}, B = {2,3,5} → A ∩ B = {2}
Q13: What is the difference of sets?
A \ B = elements in A but not in B
Example: A = {1,2}, B = {2,3,5} → A \ B = {1}
Q14: What is the complement of a set?
Aᶜ = elements in U not in A
Example: A = {1,2}, U = {0,1,2,3,4,5} → Aᶜ = {0,3,4,5}
Q15: What is the universal set?
U = the set of all possible elements in a given context. Example: for integers, U = Z
Q16: What is the empty set?
∅ = a set that contains no elements
Q17: What does A = B mean?
A = B → sets A and B are equal; they contain exactly the same elements
